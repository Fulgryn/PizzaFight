### Damn, that's now a REAL fake game!

#### Player 1 controls:
* `Z, Q, S, D` => jump, left, down, right (in order)
* `T, Y, U` => shoot!
* `G` => shield!

#### Player 2 controls:
* `Up, Left, Down, Right (arrows)` => jump, left, down, right (in order)
* `1, 2, 3 (numpad)` => shoot!
* `0 (numpad)` => shield!
